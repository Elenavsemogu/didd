# DiDD — Dog in Double Despair (сайт группы)

## Описание
Одностраничный сайт для панк-блюз / пост-хардкор группы из Санкт-Петербурга.
Секции: Hero (последний релиз), Shotgun-анимация, Tracklist, About, Releases, Merch, Footer.

## Стек
- Чистый HTML/CSS/JS (один файл `index.html`)
- Google Fonts: Space Mono + Inter
- Без фреймворков, без сборщиков

## Контент
- **Альбом**: "Hope You Never Meet Each Other" — выход май 2026, pre-save
- **Сингл**: "Cab Driver" — уже выпущен
- **О группе**: единственная международная группа России из СПб, Punk-Blues / Post-Hardcore
- **Соцсети**: Instagram + Telegram (placeholder-ссылки)
- **Контакт в подвале**: placeholder email

## Файлы
- `index.html` — основной сайт (v2, с правильным контентом)
- `index-v1.html` — бэкап первой версии дизайна
- `album-cover.png` — оригинал обложки (6MB)
- `album-cover-web.jpg` — сжатая обложка для веба (200KB)
- `album-cover-alt.jpg` — обложка альбома (вариант)
- `band-photo-1.jpg` — обложка сингла Cab Driver
- `band-photo-2.jpg` — промо-дизайн Cab Driver
- `band-logo.jpg` — арт с лотосом (используется в About)
