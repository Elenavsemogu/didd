Promo Layout — Предложения по дизайну

4 варианта расположения элементов для промо-картинки сингла/альбома в ртутной рамке. Элементы: обложка-патрон DiDD, название альбома, название трека, QR-код, подпись "on all streaming services".

Оригинал Антона (референс)
Layout A
Classic Center
Патрон по центру под углом. Название альбома сверху, трек + QR снизу. Чистая симметрия, всё читается сразу.
  • Название альбома — крупно сверху по центру
  • Патрон — центр композиции, наклон ~20°
  • Название трека — под патроном
  • QR-код — правый нижний угол
  • "On all streaming services" — мелко под треком

Layout B
Asymmetric Stack
Текст слева стопкой (альбом → трек → стриминг), патрон справа крупно. QR внизу слева с пояснением. Динамичная асимметрия.
  • Левый столбец: крупный заголовок альбома, трек красным, стриминг серым
  • Патрон — справа, занимает ~40% площади
  • QR — крупнее, внизу слева с меткой "Scan to listen"
  • Больше воздуха между элементами

Layout C
Cinema Bands
Патрон в центре чисто. Инфо в полупрозрачных полосах сверху и снизу как титры в кино. Минималистично, вся рамка видна.
  • Верхняя полоса: название альбома слева, трек справа
  • Патрон — чистый центр, максимально виден
  • Нижняя полоса: стриминг слева, QR справа

Layout D
Poster Style
Патрон сверху-центр, вся типографика крупным блоком внизу. Как плакат для концерта — дерзко, читаемо издалека.
